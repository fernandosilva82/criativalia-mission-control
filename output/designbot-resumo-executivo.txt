📸 4. GUIA DE FOTOGRAFIA DE PRODUTO (Como tirar fotos que vendem)
═══════════════════════════════════════════════════════════════════════

Setup Essencial:
  • 2 softboxes LED 40x40cm (investimento ~R$150-300)
  • Tripé (fundamental para consistência)
  • Refletor (placa de isopor ou foam board)
  • Fundo branco infinito (papel kraft ou sweep)

Iluminação Padrão (2 luzes):
  → Luz Key: 45° à esquerda, 100% potência
  → Luz Fill: 45° à direita, 50-70% potência
  → Relação ideal: 2:1

Configurações da Câmera:
  • ISO: 100-200
  • Abertura: f/8 a f/11
  • Velocidade: 1/60s a 1/125s (com tripé)
  • Formato: RAW para edição

Ângulos Obrigatórios por Produto:
  1. Frontal 0° (identificação)
  2. 45° frontal (dimensões)
  3. Lateral 90° (perfil)
  4. Superior (forma acesa)
  5. Detalhe (textura/acabamento)
  6. Lifestyle (contexto de uso)
  7. Escala (objeto de referência)

🔧 5. OTIMIZAÇÕES PARA MODELAGEM 3D (Mais rápido + Econômico)
═══════════════════════════════════════════════════════════════════════

Otimizações Estruturais:
  ✓ Paredes: 2-3 perímetros (0.8-1.2mm)
  ✓ Infill: 15-20% para luminárias (Grid ou Gyroid)
  ✓ Modelos ocos: Economia de 40-60% de material
  ✓ Evitar overhangs >45° (reduz suportes em 20-30%)

Orientação de Impressão:
  ✓ Menor altura Z possível
  ✓ Maior face plana como base
  ✓ Superfícies visíveis para CIMA

Configurações Recomendadas:
  ✓ Altura de camada: 0.20mm (padrão) ou 0.12mm (detalhes)
  ✓ Paredes externas: 30-40mm/s
  ✓ Preenchimento: 60-80mm/s

Materiais Sugeridos:
  • Corpo das luminárias: PETG (resistência + translucidez)
  • Peças decorativas: PLA premium
  • Difusores: PETG translúcido ou resina

Design para Impressão 3D (DfAM):
  • Use nervuras internas ao invés de paredes grossas
  • Tolerâncias: 0.1-0.2mm para encaixe pressão
  • Filtros internos: Grade 45° com abertura 5-10mm
