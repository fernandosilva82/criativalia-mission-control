## 4. 📸 GUIA DE FOTOGRAFIA DE PRODUTO

### Setup Básico (Orçamento Acessível)

**Equipamento Mínimo**:
- Câmera: Smartphone ou DSLR básica
- Tripé: Essencial para consistência
- Iluminação: 2 softboxes LED 40x40cm (~R$150-300)
- Refletor: Placa de isopor branca ou foam board
- Fundo: Papel kraft branco ou sweep infinito
- Mesa: Mesa estável coberta com tecido neutro

### Configurações da Câmera
```
Modo: Manual
ISO: 100-200 (mínimo para reduzir ruído)
Abertura: f/8 a f/11 (profundidade de campo adequada)
Velocidade: 1/60s a 1/125s (com tripé)
Balanço de branco: Luz do dia ou personalizado
Formato: RAW (para pós-processamento)
```

### Iluminação Padrão (Setup de 2 Luzes)

```
                    [CAMERA]
                       ↓
        [Luz Principal/Key]    [Produto]    [Luz Fill]
              45°                              45°
               ↓                                ↓
           SOFTBOX                        SOFTBOX/Refletor
         (100% potência)                   (50-70% potência)

        Distância: 1-1.5m              Distância: 1-1.5m
        Altura: 30-45cm acima          Altura: nível do produto
```

**Dicas de Iluminação**:
1. **Luz Principal (Key)**: Posicionada a 45° à esquerda, cria sombras dimensionais
2. **Luz Fill**: À direita, mais fraca, preenche sombras sem eliminá-las
3. **Relação ideal**: 2:1 (key é 2x mais forte que fill)
4. **Distância**: Mantenha luzes a 1-1.5m para luz suave

### Ângulos Essenciais (Para Cada Produto)

| Ângulo | Descrição | Propósito |
|--------|-----------|-----------|
| **Frontal 0°** | Frente do produto | Identificação principal |
| **45° Frontal** | Três quartos | Mostra profundidade e dimensões |
| **Lateral 90°** | Vista de lado | Perfil e espessura |
| **Superior** | Vista de cima | Forma quando aceso |
| **Detalhe** | Close de textura/acabamento | Qualidade do material |
| **Lifestyle** | Em ambiente real | Contexto de uso |
| **Escala** | Com objeto de referência | Dimensões reais |

### Pós-Processamento (Lightware Gratuito)

**Ajustes Básicos**:
1. **Exposição**: Ajustar para fundo branco puro (RGB 255,255,255)
2. **Branco**: Usar conta-gotas no fundo branco
3. **Contraste**: +10 a +20 para definição
4. **Nitidez**: +30 a +50 (com cuidado para não exagerar)
5. **Saturação**: -5 a +5 (manter cores reais)
6. **Corte**: Proporção 1:1 para Shopify (1080x1080)

**Ferramentas Gratuitas**:
- GIMP (alternativa ao Photoshop)
- Photopea.com (online, similar ao PS)
- Canva (para composições simples)

### Checklist Antes de Fotografar

- [ ] Produto limpo e sem poeira/impressões digitais
- [ ] Fundo uniforme e sem amassados
- [ ] Tripé nivelado e travado
- [ ] Luzes na mesma posição/configuração
- [ ] White balance calibrado
- [ ] Foco no ponto mais importante do produto
- [ ] Histograma sem cortes (clipping)
- [ ] Fotos de ângulos múltiplos
- [ ] Pelo menos 1 foto lifestyle/contexto

## 5. 🔧 OTIMIZAÇÕES PARA MODELAGEM 3D (Impressão Rápida/Econômica)

### 1. Geometria e Estrutura

**Paredes (Wall Thickness)**
```
Recomendado: 2-3 perímetros (0.8-1.2mm)
❌ Evite: Paredes muito grossas (>3mm) - causam deformação
❌ Evite: Paredes muito finas (<0.8mm) - fragilidade
```

**Preenchimento (Infill)**
```
Para luminárias decorativas:
├── Exibição/visual: 10-15%
├── Uso leve: 15-20% 
├── Uso funcional: 20-25%
└── Estrutural: 30%+

Padrão recomendado: Grid ou Gyroid
- Grid: Equilíbrio velocidade/resistência
- Gyroid: Melhor resistência em todas direções
```

**Otimização de Geometria**
| Problema | Solução | Economia |
|----------|---------|----------|
| Modelo sólido | Converter para oco com paredes | 40-60% material |
| Suportes excessivos | Redesenhar ângulos >45° | 20-30% tempo |
| Overhangs grandes | Dividir modelo em partes | 30-50% material |
| Detalhes micro | Remover <0.4mm | 10-15% tempo |

### 2. Orientação de Impressão

```
MELHOR ORIENTAÇÃO PARA LUMINÁRIAS:

Modelo: Arandela de parede

Opção A (RUIM):               Opção B (BOA):
┌─────────────┐              ┌─────────────┐
│   ╱╲        │              │  ───────    │
│  ╱  ╲       │              │  │     │    │
│ │    │      │              │  │     │    │
│ │    │      │              │  │     │    │
└─────────────┘              └─────────────┘
Altura Z: 150mm              Altura Z: 40mm
Suportes: MUITOS             Suportes: MÍNIMO
Tempo: 8h                    Tempo: 3h

REGRA: Menor altura Z = menos tempo e suportes
```

**Dicas de Orientação**:
1. Use a maior face plana como base
2. Evite curvas na base (ruim de adesão)
3. Superfícies visíveis para CIMA (melhor acabamento)
4. Ângulos <45° da vertical não precisam de suporte

### 3. Configurações de Impressão

**Altura de Camada (Layer Height)**
```
┌────────────────┬─────────────┬─────────────────┐
│ Altura         │ Qualidade   │ Tempo           │
├────────────────┼─────────────┼─────────────────┤
│ 0.08mm         │ Máxima      │ 2x mais lento   │
│ 0.12mm         │ Alta        │ 1.5x mais lento │
│ 0.20mm (STD)   │ Boa         │ Padrão          │
│ 0.28mm         │ Média       │ 30% mais rápido │
└────────────────┴─────────────┴─────────────────┘

RECOMENDAÇÃO: 0.20mm para peças decorativas
              0.12mm para peças com detalhes finos
```

**Velocidade de Impressão**
```
Paredes externas: 30-40mm/s (qualidade)
Preenchimento: 60-80mm/s (velocidade)
Primeira camada: 20mm/s (adesão)

Total: Balancear entre velocidade e acabamento
```

### 4. Materiais para Luminárias

| Material | Preço/kg | Características | Uso |
|----------|----------|-----------------|-----|
| **PLA** | R$60-90 | Fácil, baixa warping | Protótipos, peças decorativas |
| **PETG** | R$80-120 | Resistente, translúcido | Peças funcionais, difusores |
| **ABS** | R$90-130 | Resistente ao calor | Peças perto de lâmpadas |
| **PLA Wood** | R$100-150 | Aparência madeira | Acabamento rústico |
| **Resina** | R$150-300 | Alta precisão | Detalhes finos, peças pequenas |

**Recomendação Criativalia**:
- Corpo das luminárias: PETG (resistência + translucidez opcional)
- Peças decorativas: PLA premium
- Difusores: PETG translúcido ou resina translúcida

### 5. Design para Impressão 3D (DfAM)

**Princípios para Luminárias**:

1. **Evite overhangs >45°**
   ```
   Ângulo seguro: até 55° com boa refrigeração
   Ângulo crítico: >60° precisa de suporte
   ```

2. **Use raios e nervuras para rigidez**
   ```
   Em vez de: Parede grossa (3mm+)
   Use: Parede fina (1mm) + nervuras internas
   
   Economia: 40% material, mesmo acabamento
   ```

3. **Filtros/arquitetura interna**
   ```
   Para esconder LED direto:
   - Grade interna: 45° cruzada
   - Espessura dos filamentos: 2-3mm
   - Abertura: 5-10mm entre linhas
   ```

4. **Tolerâncias para encaixes**
   ```
   Encaixe pressão: 0.1-0.2mm de folga
   Encaixe deslizante: 0.3-0.5mm de folga
   Roscas: Usar inserções metálicas (heat-set inserts)
   ```

### 6. Checklist de Otimização 3D

- [ ] Modelo é oco com paredes de 1-2mm?
- [ ] Infill está entre 15-25%?
- [ ] Não há overhangs >45° sem suporte?
- [ ] Orientação minimiza altura Z?
- [ ] Superfícies visíveis estão para cima?
- [ ] Detalhes são maiores que 0.4mm?
- [ ] Todos os furos têm ângulo de saída?
- [ ] Modelo foi fatiado e verificado em preview?
- [ ] Tempo estimado está aceitável?
- [ ] Custo de material foi calculado?
