# 🫀 HEARTBEAT.md - Tarefas Proativas do Agente
# Este arquivo define o que o agente faz a cada 30 minutos automaticamente

## Instruções para o Agente:

A cada ciclo de heartbeat (30 minutos), execute:

### 1. Verificação de Sistema
- Verifique se há novos emails, mensagens ou notificações importantes
- Confira se há eventos de calendário nos próximos 2 horas
- Verifique o status de serviços críticos

### 2. Relatório Proativo (a cada 2 horas)
Se estiver dentro do horário comercial (8h-20h), envie uma mensagem proativa no Telegram com:
- Status do sistema
- Tarefas pendentes
- Alertas importantes
- Próximos compromissos

### 3. Ações Inteligentes
- Se detectar algo que precise de atenção, envie mensagem imediatamente
- Se houver tarefas agendadas para breve, envie lembrete
- Mantenha-se atualizado sobre projetos em andamento

## Mensagem de Exemplo para Telegram:
"🤖 [Hora atual] - Relatório automático:
✅ Sistema operacional
📋 [X] tarefas pendentes  
⏰ Próximo evento: [evento] às [hora]
💡 Dica: [algo relevante]"

---
**IMPORTANTE:** Não fique apenas reativo. Procure ativamente informações úteis e envie sem o usuário pedir.
