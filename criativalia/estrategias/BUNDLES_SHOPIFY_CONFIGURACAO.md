## 🛠️ IMPLEMENTAÇÃO NO SHOPIFY

### Opção 1: Shopify Bundles (Nativo)
**Grátis para Shopify Plus, Advanced, ou com app**

1. **Admin Shopify → Produtos → Bundles**
2. **Clicar "Create bundle"**
3. **Preencher:**
   - Título do bundle
   - Descrição
   - Produtos componentes
   - Preço do bundle
   - Imagem
4. **Salvar e publicar**

### Opção 2: App "Bundler" (Recomendado)
**Grátis até 100 bundles/mês**

1. **Apps → Visit Shopify App Store**
2. **Pesquisar "Bundler - Product Bundles"**
3. **Instalar**
4. **Criar bundle:**
   - Nome: "Kit Quarto dos Sonhos"
   - Produtos: Selecionar da loja
   - Desconto: Fixo de R$128 ou %
   - Imagem: Upload

### Opção 3: Desconto Automático (Sem app)
**Usar código de desconto + instruções**

1. **Criar página de produto "virtual"**:
   - Título: "Kit Quarto dos Sonhos"
   - Preço: R$499
   - Descrição explicando os itens

2. **Criar código de desconto** (uso único por compra):
   - Código: QUARTO20
   - Desconto: 20% OFF quando compra os 3 itens
   - Restrição: Mínimo 3 itens específicos

3. **Instruções no checkout:**
   - "Adicione os 3 itens ao carrinho"
   - "Use o código QUARTO20 no checkout"
