## 📸 4. GUIA DE FOTOGRAFIA DE PRODUTO

### Checklist de Fotos por Produto

**Obrigatórias (Mínimo 5):**

2. ✅ **45° Angle** - Mostrando profundidade e dimensões
3. ✅ **Detalhe** - Textura, acabamento, conexões
4. ✅ **Contexto/Lifestyle** - Produto em ambiente real de uso
5. ✅ **Escala** - Próximo a objeto conhecido (mão, livro, copo)

**Diferenciadas (Plus):**
6. 📸 **Funcionalidade** - Produto ligado/iluminado
7. 📸 **Variações** - Cores/tamanhos disponíveis
8. 📸 **Instalação** - Produto montado na parede (arandelas)
9. 📸 **Empacotamento** - Unboxing experience

### Configurações da Câmera (Smartphone Pro)

| Configuração | Valor | Por quê? |
|--------------|-------|----------|
| ISO | 100-200 | Mínimo ruído |
| Velocidade | 1/60s+ | Evitar tremida |
| Abertura | f/2.8-f/4 | Boa profundidade |
| WB | 5500K (luz dia) | Cores reais |
| Formato | RAW ou HEIF | Mais dados para editar |

### Regras de Composição

**Regra dos Terços:**
Posicione o produto nos pontos de interseção, não no centro exato.

**Ângulos Essenciais:**
- **Eye-level (nível dos olhos):** 90% das fotos de produto
- **45°:** Melhor ângulo para mostrar profundidade
- **Top-down:** Para produtos planos ou vista de cima
- **Worm's eye:** De baixo para cima - dramático

### Pós-Processamento (Edição)

**Ajustes Essenciais:**
1. **Nitidez:** +10 a +20
2. **Brilho:** Ajustar para realçar detalhes
3. **Contraste:** +5 a +15 (sutil)
4. **Saturação:** Máx +10 (manter realismo)
5. **Cortar:** Proporção 1:1 (Instagram) ou 4:5 (feed)

**O que EVITAR:**
- ❌ Oversaturação de cores
- ❌ Filtros pesados
- ❌ Bordas artificiais
- ❌ Reflexos de luz na lente
- ❌ Sombras duras (use difusor)

## ⚙️ 5. OTIMIZAÇÕES PARA IMPRESSÃO 3D
### Mais Rápida + Mais Econômica

### Resumo de Economia Potencial

| Otimização | Economia Material | Economia Tempo | Qualidade |
|------------|-------------------|----------------|-----------|
| Infill Gyroid 15-20% | -30% | -20% | ↑ Melhor |
| Paredes variáveis | -25% | -15% | = Mantida |
| Suportes otimizados | -40% (material) | -30% (pós) | = Mantida |
| Orientação estratégica | -20% | -10% | ↑ Melhor |

### Configurações Otimizadas (Bambu/OrcaSlicer)

#### Para Arandelas (Estruturais)

```
LAYER HEIGHT: 0.2mm (qualidade) / 0.28mm (velocidade)
WALLS: 3 perimeters (1.2mm) - reforço estrutural
TOP/BOTTOM: 4 camadas
INFILL: Gyroid 20-25%
INFILL PATTERN: Gyroid (isotrópico, melhor resistência)
SUPPORTS: Tree (auto) - apenas quando necessário
PRINT SPEED: 150mm/s (qualidade) / 200mm/s (rápido)
ACCELERATION: 5000mm/s²
```

#### Para Difusores/Luminárias (Estéticas)

```
LAYER HEIGHT: 0.12mm (super qualidade)
WALLS: 2 perimeters (0.8mm)
TOP/BOTTOM: 3 camadas
INFILL: Gyroid 15% ou Lightning 10%
PRINT SPEED: 100mm/s
IRONING: ON (para superfície superior lisa)
```

### Técnicas Específicas por Tipo de Peça

#### 1. Paredes Variáveis (Variable Wall Thickness)

**Onde aplicar:**
- Zonas de fixação na parede: 4 perimeters (1.6mm)
- Corpo principal: 2-3 perimeters (0.8-1.2mm)
- Bordas decorativas: 2 perimeters + arredondamento

**Como fazer no OrcaSlicer:**
1. Clique direito na peça → "Add modifier"
2. Selecione "Wall thickness"
3. Defina regiões com espessuras diferentes

**Economia:** 20-30% de material

#### 2. Infill Adaptativo (Adaptive Infill)

**Configuração recomendada:**
```
Tipo: Adaptive Cubic ou Gyroid
Densidade mínima: 10%
Densidade máxoria: 40%
Threshold: Regiões com overhang > 45°
```

**Resultado:** Material apenas onde a estrutura precisa

#### 3. Suportes Tipo Árvore (Tree Supports)

**Vantagens sobre suportes normais:**
- Menos material desperdiçado (-40%)
- Remoção mais fácil e limpa
- Menos marcas na peça final
- Tempo de impressão similar

**Configuração:**
```
Type: Tree (auto)
Style: Branch
Branch angle: 45°
Branch distance: 2mm
```

#### 4. Orientação de Impressão

**Regra de ouro:** Menor área de suporte = menos pós-processamento

| Peça | Orientação Ideal | Por quê? |
|------|------------------|----------|
| Arandela circular | Deitada (face plana na mesa) | Melhor acabamento visual |
| Base do abajur | Vertical | Evita suportes internos |
| Cúpula/difusor | Deitada | Menos layer lines visíveis |
| Pendentes | Ângulo 45° | Balanceamento suportes vs acabamento |

### Checklist Pré-Impressão

```
□ Análise de stress aplicada? (identificar zonas críticas)
□ Paredes variáveis configuradas?
□ Infill Gyroid 15-25% selecionado?
□ Suportes Tree ativados?
□ Orientação otimizada?
□ Layer height adequado ao uso?
□ Ironing habilitado para superfícies visíveis?
```
