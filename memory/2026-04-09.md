## 07:38 - WhatsApp Rotator System (Projeto para Fernando)

### Contexto
Fernando solicitou um sistema de disparo WhatsApp para mailing grande com múltiplas instâncias e rotação automática.

### Solução Criada
**WhatsApp Rotator** - Sistema completo com:

#### Estrutura
```
whatsapp-rotator/
├── public/index.html     # Interface web dark mode
├── server.js             # API + WebSocket + Web Server
├── src/core.js           # Lógica de rotação
├── docker-compose.yml    # Evolution API + Rotator
└── deploy.sh             # Setup automático
```

#### Funcionalidades Implementadas
- ✅ **Múltiplas instâncias WhatsApp** via Evolution API
- 🔄 **Rotação automática** (round-robin / least-used / random)
- 📧 **Upload CSV** de mailing
- 📊 **Dashboard** com stats em tempo real
- 📜 **Logs ao vivo** via WebSocket
- 🛡️ **Anti-block**: delay 5-10s, cooldown, limite diário

#### Configurações de Rotação
```env
ROTATION_MODE=round-robin
MESSAGES_PER_INSTANCE=100
COOLDOWN_BETWEEN_INSTANCES=5
DAILY_LIMIT_PER_INSTANCE=200
```

#### Deploy
```bash
cd whatsapp-rotator
bash deploy.sh
# Acesso: http://localhost:3333
```

#### Fluxo de Uso
1. Criar instâncias → Conectar QR Code
2. Upload CSV (telefone,nome)
3. Escrever mensagem ({nome} para personalizar)
4. Iniciar disparo → Acompanhar em tempo real

### Notas Técnicas
- Usa Evolution API (Baileys/WA Web)
- SQLite para persistência
- WebSocket para updates em tempo real
- Docker Compose para fácil deploy
- Interface responsiva (mobile-friendly)
